## sarafu-network-protocol

> Core contracts for The Sarafu Network protocol

Includes smart contracts around:

* Community asset vouchers
* Commitment pooling
* Protocol fees

### Requirements

* [Foundry toolchain](https://getfoundry.sh/introduction/overview)
